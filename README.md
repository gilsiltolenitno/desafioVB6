# Desafio VB6 - Sistema de Transações

Projeto desenvolvido em Visual Basic 6 com integração SQL Server.

## Funcionalidades
- Cadastro de clientes
- Registro de transações
- Relatório por mês

## Tecnologias
- VB6
- SQL Server

## Objetivo
Projeto criado para resoover desafio relacionado a uma prática de desenvolvimento desktop.


## Estrutura do Projeto

- /database → Scripts do banco de dados
- /relatorios → Relatórios em Excel
- Código fonte VB6 na raiz

## Banco de Dados

Executar o script:

/database/scriptstestetecnico.sql

## Observações

O arquivo Excel contém exemplo de relatório gerado pelo sistema.
